Connect a Google Sheet

Use a Google Sheet as the live data source. For the most reliable connection, in Google Sheets choose File → Share → Publish to web → Comma-separated values (.csv) and paste that link below. A normal "anyone with the link can view" URL works too.

The sheet's first row must be headers: Date, Brand, Campaign, Network, Impressions, Clicks, Spend, Conversions, Revenue (any order, case-insensitive).
